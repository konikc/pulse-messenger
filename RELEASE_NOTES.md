# Pulse v0.1.0 — первый предварительный релиз

Pulse выходит на Android, Windows, Linux и Web.

## Что включено

- личные чаты и уникальные username;
- сообщения, реакции, изображения, документы и голосовые файлы;
- аудио- и видеозвонки через LiveKit;
- адаптивный интерфейс для мобильных и настольных устройств;
- синхронизация через Neon Postgres;
- приватное хранение пользовательских файлов в Vercel Blob;
- автоматическая проверка новых GitHub Releases.

## Файлы

- `Pulse-Android-v0.1.0.apk` — подписанный APK для Android;
- `Pulse_*_x64-setup.exe` — установщик Windows;
- `Pulse_*.AppImage` — переносимое приложение Linux;
- `Pulse_*.deb` — пакет Ubuntu/Debian;
- `SHA256SUMS.txt` — контрольные суммы всех файлов.

## Важно

Это предварительная версия для тестирования. Сообщайте о найденных ошибках через GitHub Issues. На Windows и Linux первая сборка может показать предупреждение системы, потому что релиз пока не подписан коммерческим сертификатом разработчика.

# Pulse v0.1.0 — first pre-release

Pulse is now available for Android, Windows, Linux, and the Web.

This build includes direct conversations, unique usernames, reactions, private media uploads, voice files, LiveKit audio/video calls, responsive layouts, Neon synchronization, and GitHub release update checks.

This is an early testing build. Please report issues through GitHub Issues. Windows and Linux may show an operating-system warning because the desktop packages are not yet signed with a commercial code-signing certificate.
